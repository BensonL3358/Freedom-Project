Console Programming

Console Programming is essentially just programming a console. A console is a combination of any input, for example a keyboard, and an output, like a monitor. The output can be programmed with the input to create endless possibilities.

I chose to study Console Programming because it seems that the majority of socitey seem to use technology without wanting to know how it operates or the detail behind it's device.
